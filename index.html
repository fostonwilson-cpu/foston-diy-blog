Incident Response Essentials

Confirm the fundamentals before escalating or closing a ticket.

Access & Security

  • Verify least-privilege admin access before making system changes.
  • Confirm MFA and conditional access rules are active for impacted accounts.
  • Document any elevated permissions and schedule immediate revocation.

Environment Checks

  • Confirm backups and restore points are current.
  • Snapshot affected virtual machines before applying changes.
  • Validate monitoring alerts and ticket references are linked.

User Communication

  • Confirm primary and backup contacts for each department.
  • Draft status updates for Slack, email, and intranet notices.
  • Set expectations for follow-up and satisfaction surveys.
